# МІА · 8 років — Котобажання

Мобільний вішліст подарунків на день народження. Статичний сайт (HTML/CSS/JS), без збірки — можна публікувати через GitHub Pages.

## Що вже готово

- Верстка каркасу, мобільна версія + десктоп
- 10 подарунків-плейсхолдерів у `script.js` (масив `ITEMS`)
- Логіка бронювання: клік → миттєво позначається зайнятим на екрані, і тихо (без переходу зі сторінки) відправляється запис у Google Форму
- При завантаженні сторінка читає опубліковану Google Таблицю, щоб позначити вже зайняті подарунки для всіх відвідувачів

## Що треба зробити тобі перед запуском

### 1. Реальний список подарунків

Відредагуй масив `ITEMS` у `script.js` — для кожного подарунка:
- `name` — назва
- `note` — короткий підпис (необов'язково)
- `photo` — шлях до фото (поклади файли в `assets/` і onview зміни там)
- `storeUrl` — посилання на магазин

### 2. Google Форма (щоб бронювання спрацьовувало)

1. Зайди на [forms.google.com](https://forms.google.com) → створи нову форму.
2. Додай одне текстове поле, наприклад «ID подарунка».
3. Відкрий цю форму на відправку (Send/Надіслати) → вкладка `<>` (Embed HTML) — там у коді `<iframe>` побачиш посилання виду:
   `https://docs.google.com/forms/d/e/XXXXXXXXXXXXXXXX/viewform`
   Заміни в кінці `viewform` на `formResponse` — це і є `FORM_ACTION_URL`.
4. Щоб дізнатись ID поля (`entry.XXXXXXXXX`): відкрий форму для заповнення, введи щось у поле «ID подарунка», але перед відправкою відкрий інструменти розробника (F12) → вкладку Network, натисни «Надіслати», знайди запит `formResponse` і подивись його параметри — там буде `entry.1234567890=твій_текст`. Це значення встав у `FORM_ITEM_ENTRY`.
5. Встав обидва значення в `CONFIG` на початку `script.js`.

### 3. Google Таблиця (щоб усі бачили, що вже зайнято)

1. У Google Формі відкрий вкладку «Відповіді» → натисни іконку Sheets, щоб створити пов'язану Таблицю.
2. У Таблиці: `Файл → Оприлюднити в інтернеті` → обери аркуш з відповідями → формат `CSV` → Оприлюднити.
3. Замість посилання, яке дасть Google у цьому діалозі, використай надійніший **gviz**-ендпоінт (він краще працює з cross-origin запитами з іншого сайту):
   ```
   https://docs.google.com/spreadsheets/d/ТВІЙ_ID_ТАБЛИЦІ/gviz/tq?tqx=out:csv&sheet=Form Responses 1
   ```
   `ТВІЙ_ID_ТАБЛИЦІ` — це довгий рядок з посилання на таблицю. Назву аркуша (`sheet=...`) подивись на вкладці внизу таблиці (зазвичай `Form Responses 1`).
4. Встав це посилання в `SHEET_CSV_URL`.

**Важливо:** оновлення в опублікованій таблиці можуть з'являтися із затримкою (від секунд до кількох хвилин) — тому теоретично двоє людей можуть встигнути забронювати один подарунок в одне й те саме вікно часу. Для цього списку (~10 позицій, невелика кількість гостей) це прийнятний ризик.

### 4. Ілюстрації

Зараз у шапці — проста SVG-заглушка кота-єдинорога («пушинокінь»). Якщо є власні ілюстрації в стилі Пушина — заміни `<svg>` в `index.html` (секція `.hero-mascot`) на `<img src="assets/...">`.
